# Agent Naming Guide

How to name a new agent for the Tonone team.

## Rules

1. **One word, 1-2 syllables.** Short names stick. The current team averages 1.3 syllables.
2. **Physical/tangible metaphor.** Forge, Spine, Volt — things you can feel. Not abstract nouns.
3. **No shared endings.** Each name stands alone. No -er/-or suffixes, no rhyming pairs.
4. **Evokes the domain.** The name should trigger the right mental image before you read the description.
5. **No trademark collisions.** Avoid names of existing tech products (Vault → HashiCorp, Sentry → sentry.io, Swift → Apple, etc.).
